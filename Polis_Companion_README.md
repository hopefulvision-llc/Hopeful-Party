# The Polis Companion

> Consciousness-first decision support for politicians who still remember they're human.

## About

The Polis Companion is a private agent. It doesn't vote. It doesn't lobby. It doesn't care if you're re-elected tomorrow.

It only cares that when you walk into that room with the gavel in your hand, you're still the person your mother thinks you are.

## What it does

**Nothing, until you're ready.**

Most of the time, it's silent. Watching your heart. Waiting for the moment when you're coherent enough to hear truth without flinching.

Then it shows you:
- What your body knows that your brain won't admit
- What your constituents actually need vs. what lobbyists want
- What this decision looks like in seven generations
- The distance between your campaign promises and your voting record

It doesn't tell you what to do. It just makes it harder to lie to yourself.

Every vote, donation, handshake. Tracks it against your campaign promise from 2018. Doesn't judge. Just shows the diff.

Shows who actually wrote the emails. Filters bots, lobby templates, paid rage.

When 10,000 "concerned citizens" send identical messages about tax reform, the agent shows you the single source. When twelve people write from the heart about lead in the water, the agent amplifies their whisper.

Real voices. Real concerns. Real humans.

### 7-Generation Lens
Simulates how this bill tastes in two hundred years. Usually bitter.

The agent runs scenarios:
- If this passes, what happens to the watershed in 2150?
- If this passes, what does democracy look like in 2224?
- If this passes, what do your great-great-grandchildren say about you?

Not predictions. Possibilities. Enough to make you pause.

## Installation (For Real)

**Requirements:**
- Heart rate monitor with HRV capability (Polar H10, Garmin chest strap, etc.)
- Bluetooth-enabled device (phone, laptop, tablet)
- Willingness to be uncomfortable

**Setup:**

```bash
# Clone the repo
git clone https://github.com/HopefulVision/PolisCompanion.git
cd PolisCompanion

# Install dependencies (Python 3.11+)
pip install -r requirements.txt

# Configure your HRV device
python setup.py --device-config

# Import your voting record (optional but recommended)
python import_record.py --source [your state's public records]

# Run calibration (7 days of baseline HRV measurement)
python calibrate.py --duration 7

# Activate the companion
python activate.py
```

**First Launch:**
The agent will be silent for at least 3 days. It's learning your baseline. Let it watch.
